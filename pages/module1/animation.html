Анимация в CSS (animation)

animation

animation: none 0s ease 0s 1 normal none running; 1. animation-name # название анимации 2. animation-duration # как долго длится анимация 3. animation-timing-function # как вычисляются промежуточные состояния 4. animation-delay # анимация начинается спустя некоторое время 5. animation-iteration-count # сколько раз должна выполняться анимация 6. animation-direction # должно движение идти в обратную сторону или нет 7. animation-fill-mode # какие стили применяются до начала анимации и после её завершения 8. animation-play-state # проигрывать анимацию или поставить её на паузу @keyframes move{ 0% { } 25% { transform: translate(300px, 0px); background: red; } 50% { transform: translate(300px, 300px); background: gray; } 75% { transform: translate(0px, 300px); background: green; } 100% { } } div { animation-name: move; animation-duration: 5s; animation-timing-function: linear; animation-delay: 0s; animation-iteration-count: 1; animation-direction: normal; animation-fill-mode: none; animation-play-state: running; } div { border: 1px solid #000; width: 200px; height: 200px; background: orange; animation: move 5s infinite; }
