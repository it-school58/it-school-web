Списки: маркированные и нумерованные (ul, ol)

Маркированный список

<ul> <li>item1</li> <li>item2</li> <li>item3</li> </ul>

Нумерованный список

<ol> <li>item1</li> <li>item2</li> <li>item3</li> </ol>

Настройка маркеров

ul { list-style: list-style-type list-style-image list-style-position; list-style-type: none | circle ... # изменяет вид маркера list-style-image: none | url('путь к файлу') # устанавливает изображение list-style-position: inside | outside # размешение относительно текста }
