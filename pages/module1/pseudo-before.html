Псевдоэлементы (before, after)

  • before - добавляет content перед контентом блока
  • after - добавляет content после контента блока
<div>содержимое</div> div { padding: 20px; width: 400px; } div::before { content: "добавлено через before"; background: white; } div::after { content: "добавлено через after"; background: white; }
