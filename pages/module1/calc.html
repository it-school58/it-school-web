calc() - вычисление значения

  • Выражение представляет собой комбинацию значений размеров и знаков математических действий
div { width: calc(20px + 20px); # сложение width: calc(10% - 10px); # вычитание width: calc(20% * 2); # умножение width: calc(100% / 3); # деление (на ноль делить запрещено) }
