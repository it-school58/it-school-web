Таймеры

  • setTimeout() - однократное выполнение кода
const timer = setTimeout(function() { console.log('go'); }, 1000); clearTimeout(timer);
  • setInterval() - многократное выполнение кода
const timer = setInterval(function() { console.log('go'); }, 1000); clearInterval(timer);

Примеры

console.log(1); setTimeout(function() { console.log(2); }, 0); console.log(3); // => 1, 3, 2
