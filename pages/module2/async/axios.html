GET-запрос с возвратом одного значения

axios.get('script.php') .then(function(response) { console.log(response.data); // 'Hi' }) .catch(function(error) { console.log(error) }); <?php echo 'Hi'; ?>

GET-запрос с возвратом нескольких значений (array)

axios.get('script.php') .then(function(response) { console.log(response.data); // ['Jack', 25] }) <?php $arr = array("Jack", 25); $json = json_encode($arr); echo $json ?>

GET-запрос с возвратом нескольких значений (object)

axios.get('script.php') .then(function(response) { console.log(response.data); // {name: "Jack", age: 25} }) <?php $arr = array("name" => "Jack", "age" => 25); $json = json_encode($arr); echo $json ?>
