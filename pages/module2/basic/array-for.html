Массивы и циклы

Объявление массива

const arr = []; // объявление пустого массива const arr = ['Tony', 'Anne']; // объявление массива из заполненных элементов

Цикл for

  • break - выход из цикла
  • continue - прекращает выполнение текущей итерации цикла и переходит на следующую
for (let i=0; i < 10; i++) { console.log(i); }

Перебор массива через цикл for

for (let i=0; i<=arr.length-1; i++) { console.log(arr[i]); }
