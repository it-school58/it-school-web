Выборка элементов страницы

Объектная модель представления документа

  • Объектная модель представления документа. (DOM) Document Object Model
  • Независящий от платформы и языка программирования способ представления документа, позволяющий программам и скриптам получить доступ к содержимому HTML, XHTML и XML документов, а также изменять содержимое, структуру и оформление таких документов
  • Всё содержимое документа представлено в виде иерархии объектов
  • Структура этой иерархии стандартизирована, чтобы обеспечить независимость от платформы

Структура DOM документа

Выборка элементов страницы

const a = document.querySelector('.nav.bar'); // возвращает первый элемент внутри документа const a = document.querySelectorAll('.nav'); // возвращает список элементов в пределах документа const a = document.getElementById('item'); // по id const a = document.getElementsByClassName('item')[0]; // по названию класса (первый элемент) const a = document.getElementsByName('item'); // по атрибуту name const a = document.getElementsByTagName('img'); // по названию тега (все элементы)
