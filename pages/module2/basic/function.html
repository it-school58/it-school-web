Функции

Функция, не возвращающая значение

sum(34, 12); function sum(a, b) { const result = a + b; console.log(result); }

Функция, возвращающая значение

const result = sum(34, 12); function sum(a, b) { const result = a + b; return result; }

Глобальные и локальные переменные

const a = 10; // глобальная переменная function test() { const b = 20; // локальная переменная console.log(a, b); }

Пример: Определить площадь поверхности планет

Решение 1

// Площадь поверхности сферы // A = 4*pi*r*r const mars = 3390; // km const mercury = 2440; // km const pi = 3.14; const surfaceMars = 4 * pi * mars * mars; const surfaceMercury = 4 * pi * mercury * mercury;

Решение 2 (через функции)

const mars = 3390; // km const mercury = 2440; // km const pi = 3.14; function surface(r) { return 4 * pi * r * r; } const surfaceMars = surface(mars); const surfaceMercury = surface(mercury);
