# CSS Multi Columns

```css
article {
	column-count: 2;              /* кол-во колонок */
	column-width: 200px;          /* ширина колонки */
	column-gap: 20px;             /* gap между колонками */
	column-rule: 1px dotted #000; /* линия между колонками */
}
```
