# Вывод сообщения пользователю (alert, prompt, confirm)

<!------------------------------------------------------------->
#### Модальное окно
<!------------------------------------------------------------->
```js
alert('Привет Мир');
```

<!------------------------------------------------------------->
#### Модальное окно с вопросом и полем ввода
<!------------------------------------------------------------->
```js
const age = prompt('Сколько Вам лет?');
```

<!------------------------------------------------------------->
#### Модальное окно Ок/Отмена (вернет bool значение)
<!------------------------------------------------------------->
```js
const answer = confirm('Вы согласны?');
```
