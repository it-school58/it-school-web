# Подключение файлов JavaScript

<!------------------------------------------------------------->
#### Подключение на странице
<!------------------------------------------------------------->
```html
<script>
	// js code
</script>
```

<!------------------------------------------------------------->
#### Подключение в отдельном файле
<!------------------------------------------------------------->
```html
<script src="js/main.js"></script>
```

[Внешние скрипты, порядок исполнения](https://learn.javascript.ru/external-script)
